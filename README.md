# Paloma - EbitenEngine Game on Pure Go

Paloma is a cross-platform game built entirely using pure Go and the powerful Ebitengine graphics library.

#### Installation and Updates

Getting started is simple, and updates are handled automatically!

1) Download the latest binary file for your specific operating system (Windows, Linux, macOS) from the [Releases](https://github.com/sk1bid/paloma-releases/releases) section.

2) Run the executable file.

3) When new updates are released, Paloma client will automatically download and apply the necessary changes upon launch.
